``delete_theme( theme_id: str ) -> None``

Delete a theme.

## Parameters

| parameter |	type | description |
| -- | -- | -- |
| theme_id | str | Theme identification string e.g. "my_dark_theme" |

## Returns

_None_

## Raises

| type | description |
| -- | -- |
| ValueError | Theme does not exist. |
